# QBT 自定义价格段控盘取数

Chrome 浏览器扩展，自动从 Nint 平台批量检索品牌数据，支持多渠道对比和 CSV 导出。

## 功能

- **批量品牌采集**：输入品牌列表，自动逐个检索
- **双渠道对比**：自动获取"全部"和"淘宝全部"两个渠道数据
- **天猫自动计算**：天猫 = 全部 − 淘宝全部
- **智能跳过**：品牌无数据时自动跳过，不浪费时间
- **CSV 导出**：一键下载，Excel 直接打开
- **固定窗口**：Pin 按钮打开独立窗口，页面刷新不消失，自带最小化
- **快捷键**：Enter 开始采集，支持 Excel 直接粘贴品牌列

## 安装

1. 下载代码或 `git clone https://github.com/aidenLi377/QBT_Brand-Channel.git`
2. 打开 Chrome，访问 `chrome://extensions`
3. 开启右上角"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目目录

## 使用

1. 打开 Nint 目标页面（`art.nint.com/stat-ali-new?...`）
2. 点击扩展图标，在弹窗中输入品牌列表（一行一个，支持 Excel 粘贴）
3. 点击"开始采集"或按 Enter
4. 等待采集完成，点击"导出 CSV"

### CSV 输出格式

| 渠道 | 品牌名称 | 202601 | 202602 | ... |
|------|---------|--------|--------|-----|
| 全部 | 品牌A | ... | ... | ... |
| 淘宝全部 | 品牌A | ... | ... | ... |
| 天猫 | 品牌A | ... | ... | ... |

## 项目结构

```
├── manifest.json     # Chrome 扩展配置
├── background.js     # Service Worker（角标、窗口管理）
├── popup.html        # 弹窗界面
├── popup.css         # 弹窗样式
├── popup.js          # 弹窗逻辑（输入、进度、CSV 导出）
├── content.js        # 核心采集脚本（DOM 操控、表格解析）
└── icons/            # 扩展图标
```
